# InsDow - Video Downloader Application

## Overview
A web-based application to download videos from online sources.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS (Premium Design), JavaScript
- **Backend**: Node.js (Express)
- **Features**:
  - URL Input Field
  - Video Preview (if applicable)
  - Download Button
  - Support for [To Be Decided] platforms

## Structure
- `public/`: Static files (HTML, CSS, JS)
- `server.js`: Backend logic for handling downloads
- `package.json`: Dependencies
